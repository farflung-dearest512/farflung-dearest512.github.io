## What is this tool?

This application creates bootable Windows USB drives. You run it on your Mac. It replaces the need for complex virtual machines or Boot Camp assistants. You can install Windows 7, 8, 8.1, 10, or 11. It handles older Legacy BIOS systems and modern UEFI systems. The software copies your installation files and prepares the USB drive for use in a Windows computer.

## ⚙️ System requirements

Ensure your Mac meets these conditions before you begin:

*   Operating System: macOS 10.15 or later.
*   Hardware: An Intel or Apple Silicon Mac.
*   Storage: A USB flash drive with at least 8 GB of space.
*   Files: A Windows ISO image file. You can download this file from the official Microsoft website.
*   Permissions: You need administrator access to your Mac account.

## 🔌 Preparing the USB drive

This process erases all existing data on your USB stick.

1.  Connect your USB drive to a port on your Mac.
2.  Open the macos-rufus application.
3.  Select your USB drive from the drop-down menu labeled "Device."
4.  Ensure the "Boot selection" setting is set to "Disk or ISO image."
5.  Click the "Select" button.
6.  Navigate to your Windows ISO file and click "Open."

## 🚀 Creating your bootable drive

Once your settings are ready, the app handles the rest:

1.  Check the Partition scheme setting. Choose "GPT" for modern computers or "MBR" for older hardware.
2.  The software automatically splits large files if you choose to install Windows 11. This prevents errors during the installation process.
3.  Click the "Start" button.
4.  A progress bar appears at the bottom. This shows the status of the file copy process.
5.  Wait for the bar to reach 100%. The app displays a "Ready" status when finished.
6.  Eject your USB drive safely before you remove it from your Mac.

## ❓ Frequently asked questions

**Does this work with all Windows versions?**
Yes. The tool supports Windows 7, 8, 8.1, 10, and 11.

**What if my drive does not appear in the menu?**
Check your connection. Ensure the drive is formatted correctly using the Disk Utility app on your Mac. Choose "ExFAT" or "MS-DOS (FAT)" if you need to wipe the drive first.

**Does this require a virtual machine?**
No. This tool runs directly on your macOS system. It avoids the performance loss associated with virtualization.
